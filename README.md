# HuanXinTest

基于环信3.0的一个小demo，添加好友，删除好友，会话列表，好友列表，单聊，群聊，聊天，发送语音，文字，图片等等功能.

##Update Log
- 2016-5-5 初步增加了会话页面的功能，能获得当前内存中的消息。
- 2016-5-7 增加了聊天模块（目前只有文字发送，马上完善）
